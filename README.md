# Clash Royale Card Explorer 🃏

Interactive web application that allows users to explore, search, and analyze Clash Royale cards with ease. This project was built as a capstone to demonstrate frontend development,handling API calls ..

---

## Features

 **Search & Filter Cards** – Quickly find cards by name, rarity, elixir cost, or type
 **Detailed Card Stats** – Rarity Type , elixir cost, and more
 **Clean UI** – Responsive and visually appealing design
 **Mobile Friendly** – Works smoothly across devices

---

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:**  CSS
* **Data Source:** Clash Royale API / Static JSON
